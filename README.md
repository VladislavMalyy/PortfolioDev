# PortfolioDev #

## Install project ##

### GIT CLONE ###

   git clone https://github.com/VladislavMalyy/PortfolioDev.git
   
### NPM INSTALL ###

   Write in cmd "npm install"
  
### WEBPACK RUN ###

   Write in cmd "webpack"

### RUN SERVER ###

   Write in cmd "npm run dev"
  
### LOOK PROJECT ###
  
   Open browser and go to http://localhost:8090/
